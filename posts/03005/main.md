# オープンキャンパスの準備中
<a href="http://www.it-chiba.ac.jp/admissions/event/oc/201706/">6/18（日）のオープンキャンパス</a>は上田研がメインで出展することになりました。訪問した人に自律ロボット（<a href="https://lab.ueda.tech/?post=02524">このイベントでも使用した改造壁伝いねずみ</a>）を組み立ててもらうという企画の準備中です。
